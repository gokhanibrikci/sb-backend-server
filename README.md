# Team MCP Server

A centralized MCP server designed to assist the entire software team.

## Features

### For Developers
- **Git Operations**: Check status, commit changes (extensible).
- **Database**: Run queries.
- **Scaffolding**: Create new projects quickly.

### For Product Owners
- **Requirements**: Search and read documentation.

### For Testers
- **Test Management**: Run tests and save new test cases.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Build**
   ```bash
   npm run build
   ```

3. **Run**
   ```bash
   npm start
   ```

## Configuration
Edit `src/index.ts` or tool files in `src/tools/` to customize behavior or connect to real services.
